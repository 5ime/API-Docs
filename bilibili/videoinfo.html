# 哔哩哔哩视频信息 正常

# 接口地址

https://tenapi.cn/v2/bilivideo

# 请求示例

curl https://tenapi.cn/v2/bilivideo -X POST -d 'url=https://www.bilibili.com/video/BV1A5411t76w'

# 请求参数

参数名 类型 必填 说明
url string 是 视频链接
