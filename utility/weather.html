# 天气预报查询 正常

# 接口地址

https://tenapi.cn/v2/weather

# 请求示例

curl https://tenapi.cn/v2/weather -X POST -d 'city=北京'

# 请求参数

参数名 类型 必填 说明
city string 是 城市名称

# 返回参数

参数名 类型 说明
city string 城市名称
date string 日期
weather string 天气
temp int 温度
humidity string 湿度
wind string 风向
pm25 int pm2.5
pm10 int pm10
low int 最低温度
high int 最高温度
airData string 空气质量指数
airQuality string 空气质量
dateLong int 时间戳
weatherType int 天气类型
windLevel int 风力等级
province string 省份
