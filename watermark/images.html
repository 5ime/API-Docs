# 聚合短视频图集去水印 正常

TIP

支持列表: 皮皮虾, 抖音, 快手, 皮皮搞笑, 最右, 微博

# 接口地址

https://tenapi.cn/v2/images

# 请求示例

curl https://tenapi.cn/v2/images -X POST -d 'url=https://weibo.com/3669102477/MmC0IxClQ'

# 请求参数

参数名 类型 必填 说明
url string 是 图集链接

# 调用示例

移步: https://lab.5ime.cn/video 查看
