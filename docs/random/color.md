# 随机颜色 <Badge text="正常" type="tip"/>

## 接口地址

``` 
https://tenapi.cn/v2/color
```

## 请求示例

``` bash
curl https://tenapi.cn/v2/color
```

## 返回数据

``` json
{
  "code": 200,
  "msg": "success",
  "data": {
    "color": "#C3B90F"
  }
}
```

## 返回参数

| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| color | string | 随机颜色 |
