``` 
https://tenapi.cn/v2/qzoneimg
```

``` 
https://tenapi.cn/v2/qzoneimg?qq=123456
```

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| qq | int | 是 | QQ号 |

```
http://qlogo4.store.qq.com/qzone/123456/123456/100
```
