# 请求示例

# 返回参数

参数名 类型 说明
id string 歌曲id
songs string 歌曲名称
sings string 歌手名称
album string 专辑名称
cover string 歌曲封面
url string 歌曲链接
name string 评论者昵称
comment string 热评
