# 网易云MV信息 正常

# 接口地址

https://tenapi.cn/v2/wyymv?id=5439044

# 请求示例

curl https://tenapi.cn/v2/wyymv -X POST -d 'id=5439044'

# 请求参数

标红部分即为MV ID
https://music.163.com/#/mv?id=5439044

参数名 类型 必填 说明
id int 是 网易云MV ID

# 返回参数

参数名 类型 说明
id int 网易云MV ID
songs string 歌曲名称
sings string 歌手名称
time string 发布时间
cover string 封面图片
mv object MV地址
mv.240 string 清晰度 240P
mv.480 string 清晰度 480P
mv.720 string 清晰度 720P
mv.1080 string 清晰度 1080P
view int 播放量
like int 点赞量
comment int 评论量
collect int 收藏量
share int 分享量
