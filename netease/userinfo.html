# 网易云用户信息 正常

# 接口地址

https://tenapi.cn/v2/mywyy

# 请求示例

curl https://tenapi.cn/v2/mywyy -X POST -d 'uid=1'

# 请求参数

标红部分即为用户 ID
https://music.163.com/#/user/home?id=1

参数名 类型 必填 说明
uid int 是 网易云用户 ID

# 返回参数
