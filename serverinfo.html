# 服务器信息查询 正常

# 请求URL

https://tenapi.cn/serverinfo

# 请求示例 GET

https://tenapi.cn/serverinfo/?url=www.baidu.com

# 返回数据

{
  "msg":200,
  "host":"www.baidu.com",
  "ip":"180.101.49.11",
  "position":"江苏省宿迁市 电信"
}

# 调用效果

暂无

# 请求参数

参数名称 类型 参数值 描述
url 必填 url 即你所需要查询的网站
