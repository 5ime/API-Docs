# QQ匿名头像获取 正常

# 请求URL

https://tenapi.cn/qqtx

# 请求示例 GET

https://tenapi.cn/qqtx?qq=123456

# 返回数据 JSON

{
  "code": 200,
  "data": {
    "qq": "123456",
    "url": "https://q1.qlogo.cn/g?b=qq&k=xyOnRe5ML3Aw96iaaQ1hh6w&s=140"
  }
}

# 调用效果

暂无

# 请求参数

参数名称 类型 参数值 描述
qq 必填 qq 即需要获取匿名头像的QQ
